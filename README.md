# 🗄️ Banco de Dados

Repositório destinado ao armazenamento e organização das listas de exercícios e projetos desenvolvidos durante a disciplina de **Banco de Dados**.

---

### 🎯 Objetivos
* Praticar comandos de Definição de Dados (**DDL**).
* Desenvolver consultas complexas com Linguagem de Manipulação de Dados (**DML**).
* Organizar scripts SQL para versionamento e consultas futuras.

### 🛠️ Tecnologias e Ferramentas
* **SGBD:** PostgreSQL
* **Linguagem:** SQL
* **Ferramentas de Design:** brmodeloweb

### 📂 Estrutura do Repositório
* Lista02: Fundamentos de criação de tabelas e tipos de dados.
* Lista03: Exercícios de inserção e manipulação básica de registros.
* Lista04: Práticas de consultas simples e filtragem de dados.
* Lista05: Consultas avançadas e funções de agregação.
* Lista06: Relacionamentos entre tabelas e uso de JOINs.
